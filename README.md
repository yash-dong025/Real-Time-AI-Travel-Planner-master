# AI-Powered Travel Planning Agent

An intelligent travel planning application that generates personalized itineraries using AI.

## Features
- AI-generated day-wise itineraries
- Hotel recommendations
- Weather forecasts
- Interactive maps
- Packing lists
- PDF export

## Setup
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add API keys to `.env` file
4. Run: `streamlit run app.py`

## API Keys Required
- OpenAI API
- Google Maps API
- OpenWeatherMap API

## **🔑 API Keys Setup (All FREE)**

### **1. OpenAI API Key**

1. Go to: https://platform.openai.com/signup
2. Sign up with email
3. Go to: https://platform.openai.com/api-keys
4. Click "Create new secret key"
5. Copy key (starts with sk-...)


### **2. Google Maps API Key**

1. Go to: https://console.cloud.google.com/
2. Create new project
3. Enable "Maps JavaScript API" and "Places API"
4. Go to Credentials → Create API Key
5. Restrict key to Maps APIs only

### **3. OpenWeatherMap API**

1. Go to: https://openweathermap.org/api
2. Sign up for free
3. Get API key from dashboard
